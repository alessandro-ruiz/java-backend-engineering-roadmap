\# Clase 0.1 — Cómo una computadora ejecuta programas



\## Idea principal



Un programa guardado en disco no está ejecutándose. Para ejecutarse, el sistema operativo lo carga en RAM y la CPU procesa sus instrucciones.



\## Componentes principales



\### CPU



La CPU ejecuta instrucciones y operaciones.



\### RAM



La RAM guarda temporalmente datos y programas mientras están en uso.



\### Disco



El disco guarda archivos de forma permanente.



\### Sistema operativo



El sistema operativo administra recursos como CPU, RAM, disco, red, permisos y procesos.



\### Proceso



Un proceso es un programa en ejecución.
